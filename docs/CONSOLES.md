# 8-Bit Video Game Consoles (1976–1990)

### The Early Cartridge Pioneers (1976–1979)
- [ ] Fairchild Channel F (1976) - The first console to use programmable ROM cartridges.
- [ ] Atari 2600 / VCS (1977) - The definitive 6507-based system that defined the era.
- [ ] Magnavox Odyssey² (1978) - Featured a built-in membrane keyboard and "Intel 8048" CPU.
- [ ] Mattel Intellivision (1979) - The first 16-bit processor (CP1600) used in an 8-bit era architecture.

### The "Second Wave" Powerhouses (1982–1983)
- [ ] Emerson Arcadia 2001 (1982) - A short-lived Signetics 2650-based console.
- [ ] ColecoVision (1982) - Famous for near-arcade-perfect ports using the Z80A CPU.
- [ ] Atari 5200 (1982) - Consolized version of the Atari 8-bit computers.
- [ ] Casio PV-1000 (1983) - A Z80A system released primarily in Japan.
- [ ] Sega SG-1000 (1983) - Sega's first entry into the home console market.

### The Third Generation Giants (1983–1987)
- [ ] Nintendo Entertainment System / Famicom (1983) - The Ricoh 2A03 (6502-based) legend.
- [ ] Sega Mark III / Master System (1985) - A Z80-powered rival with a superior color palette.
- [ ] Atari 7800 (1986) - Featured the MARIA graphics chip for massive sprite handling.
- [ ] Atari XEGS (1987) - A repackaged 6502 computer that functioned as a console.

### Late 8-Bit & Specialized Systems (1987–1990)
- [ ] PC Engine / TurboGrafx-16 (1987) - A "hybrid" with an 8-bit CPU (HuC6280) and 16-bit graphics.
- [ ] Commodore 64 Games System (1990) - A keyboardless version of the C64 computer.
- [ ] Amstrad GX4000 (1990) - Amstrad's console attempt based on the "Plus" computer range.

### Notable 8-Bit Handhelds
- [ ] Nintendo Game Boy (1989) - Powered by a Sharp LR35902 (Z80/8080 hybrid).
- [ ] Atari Lynx (1989) - The first handheld with a backlit color LCD and 16-bit graphics (8-bit CPU).
- [ ] Sega Game Gear (1990) - Essentially a portable, battery-hungry Master System.

# 16-Bit Video Game Consoles (1987–1993)

### The Early Leaders (1987–1988)
- [ ] PC Engine / TurboGrafx-16 (1987) - An 8-bit CPU (HuC6280) paired with a powerful 16-bit Video Display Controller.
- [ ] Sega Genesis / Mega Drive (1988) - The "speed demon" powered by a Motorola 68000 at 7.67 MHz and a Z80 for sound.

### The Heavy Hitters (1990–1991)
- [ ] Super Nintendo Entertainment System / Super Famicom (1990) - Famous for "Mode 7" rotation/scaling and the Sony SPC700 sound chip.
- [ ] Neo Geo AES (1990) - Literally "arcade hardware in a home box." Extremely powerful with massive cartridges.
- [ ] Philips CD-i (1991) - An early attempt at a "multimedia" console based on the 68000.

### The CD-ROM Evolution (1991–1992)
- [ ] Sega CD / Mega-CD (1991) - An add-on that added a second 68000 CPU and hardware for sprite rotation and scaling.
- [ ] TurboDuo / PC Engine Duo (1991) - Integrated the PC Engine with its CD-ROM² peripheral.

### The Late-Era 16-Bit / "Transition" Systems
- [ ] Pioneer LaserActive (1993) - A massive LaserDisc player that could run Genesis and PC Engine games via "LD-ROM" packs.;
- [ ] Super A'Can (1995) - A rare, powerful 68000-based console released only in Taiwan.
